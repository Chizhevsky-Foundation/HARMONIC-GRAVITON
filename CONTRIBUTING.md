# Guía de Contribución - Proyecto HARMONIC-GRAVITON

¡Bienvenido! Si estás aquí, es porque crees que el universo tiene una melodía y que no necesitas una supercomputadora de última generación para escucharla.

## Cómo puedes contribuir

### 1. Optimización para Hardware Reciclado
Nuestra prioridad es que el código corra en máquinas con historia (como nuestro i7 860). Si puedes hacer que un script de análisis sea más eficiente en memoria o use mejor los hilos de CPUs antiguas, tu aporte es oro.

### 2. Análisis de Datos (Open Data CERN)
Si encuentras nuevas anomalías en los datasets de ATLAS o CMS (específicamente en los ángulos de 120° o 60°), abre un **Issue** con el título `RESONANCIA: [Tu hallazgo]`.

### 3. Validación de la Firma 1.1740
Buscamos investigadores que puedan replicar nuestros hallazgos usando el "Sello de la Fundación". Si logras validar la constante en otros datasets (LHCb, ALICE), envía un **Pull Request**.

## Proceso de Pull Request
1. Haz un **Fork** del repositorio.
2. Crea una rama para tu mejora (`git checkout -b feature/mejora-cuantica`).
3. Realiza tus cambios y asegúrate de que el código sea legible (estilo "Ciencia Clara").
4. Envía el Pull Request describiendo cómo tu cambio ayuda a la detección del Gravitón.

## Nuestra Filosofía
* **Código Limpio, Mente Clara:** Comenta tu código pensando en que otro investigador lo leerá desde una terminal antigua.
* **Respeto al Hardware:** Valoramos más la eficiencia que la fuerza bruta.

*¡Hagamos ciencia que trascienda la obsolescencia programada!*
